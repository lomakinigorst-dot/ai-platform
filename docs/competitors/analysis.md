# Анализ конкурентов — AI SaaS платформа

## 1. suvvy.ai

**Модель:** Маркетплейс AI-ботов. Пользователь выбирает готового бота из каталога.

**Что есть:**
- Каталог готовых ботов: продажник, генератор КП, HR, закупки, кулинар, фитнес и др.
- Цены: бесплатно / 300₽ / 1000₽ / 1990₽ в месяц за бота
- Разделы: Чаты, Агенты, Рассылки, История, Аналитика, Логи запросов

**Слабое место:** Нет партнёрской (агентской) модели. Клиент настраивает сам — нет сопровождения агентством. Нет единой базы знаний из сайта.

## 2. intly.ru

**Модель:** Агентство (Интегратор) управляет клиентами. 2 типа аккаунта: Клиент и Интегратор.

### Партнёрский кабинет (Интегратор)
- Раздел: Обзор, Клиенты, Доход, Выплаты, Реквизиты, Эффективная ставка
- Финансы: доход по дням, hold-период 14 дней, минимум выплаты 1000₽
- Требует ИП или ЮЛ (самозанятые не могут быть партнёрами)
- Оплата: CDEK Pay или счёт для юр. лиц

### Клиентский кабинет
**Меню:**
- AI Агент Интли
- Аналитика (диалоги, сообщения, уверенность AI, эскалации, тональность, снимок воронки)
- Мессенджер (входящие диалоги)
- Лиды (Канбан-воронка: Новые → Первый контакт → Квалифицированы → КП → Переговоры → Закрытые)
- База знаний ИИ (файлы, ссылки, изображения, Q&A, тексты + счётчик векторов)
- Настройки ИИ (AI-сотрудники, триггеры, история действий)
- Интеграции
- Продвижение ИИ (генерация постов с картинками, планировщик публикаций)

**Настройки ИИ:**
- Выбор модели: GPT-5.4 Mini, GPT OSS 120B, Gemini 3 Flash, MiMo V2 Flash (drag & drop приоритет fallback)
- Параметры: температура, макс. токены (32768), мин/макс длина ответа, top-P, штраф частоты
- Пороги эскалации: 50% (эскалация) / 70% (уточняющий вопрос) / 100% (автоответ)
- Имитация печати человека: скорость 5 симв/сек, вариативность ±20%
- Возможности агента: чтение/изменение CRM, события и напоминания, рассылки, чёрный список, аналитика воронки

**Тарификация клиентов:** по потреблению
- Простой диалог (до 5 ответов): 6₽
- Сложный диалог (6+ ответов): 12₽
- Ответ на отзыв (WB/Ozon/Яндекс): 2₽
- Рассылка: 0.50₽/сообщение

**Интеграции (38):**
- Мессенджеры (7): Telegram App, Telegram Bot, WhatsApp, MAX Messenger, MAX Personal, Wazzup, ChatPush
- Соцсети (2)
- Маркетплейсы (4)
- CRM (8): AmoCRM, Bitrix24, YClients, МойСклад, 1С, МераПлан, Platrum, YouGile
- Сервисы (7): Google Sheets, Google Drive, Google Docs, Google Sheets CSV, Google Calendar, Яндекс Диск, Яндекс Почта
- Логистика (2), Платежи (3)
- Остальные (5): Виджет Intly, JivoSite, GitHub, Unisender, RuSender

**Слабое место:** Нет HR, финансов, юриста, отдела продаж. Только AI-консультант + маркетинговые посты. Нет автосканирования сайта — знания загружаются вручную.

## 3. b24u.ru

**Модель:** Партнёрская — агентство ведёт клиентов через платный партнёрский портал.

### Партнёрский кабинет
- Уровни: Золотой $500/мес, Премьер $1000, Платиновый $2500, Алмазный $5000
- Золотой: 50 сайтов, 15 триалов, 1 из 50 клиентов доступен
- Доход реального партнёра (видно): $1373/мес, $285 расходы, $327 баланс
- ИИ Сканер: анализ сайта по URL → создание профиля компании (в рамках тарифа лимит сканов)
- Реферальная программа: 10% комиссия, cookie 90 дней
- Команда: роли Manager, Integrator, Sales Manager — детальные права
- Финансы: история транзакций с CSV-экспортом

### Ключевые функции продукта
**AI-виджет (для клиента):**
- Обогащение контекста в реальном времени: URL страницы, UTM-метки, источник трафика, JSON-LD (цены, наличие, рейтинги), пользовательский контекст через window.B24U_CONFIG
- Аналитические события: для GA/GTM/Яндекс.Метрики — b24u_chat_открыт, контакт получен, email/phone получен
- Каналы: виджет сайта, WhatsApp Business, Instagram, Messenger Facebook, Telegram Bot, VK, Avito

**Тарифы клиентов (12 планов):**
- Trial → Start AI → Smart AI → Pro AI ($70-172/мес) → Mega AI ($297/мес) → Max/Super/Ultra/Enterprise → Market 1/2/3

**Сравнение функций по планам (38+ функций):**
- История беседы, полная аналитика, интеграция GA/GTM, фид товаров (до 500 000 в Enterprise)
- Avito, Telegram компаньон, система бронирования (Google Calendar + YClients), VK/Meta/Ozon/YouTube/Email помощники
- CRM встроенный + интеграции внешних CRM

**Рекомендации — 24-шаговое руководство онбординга:**
1. Получить адрес сайта → проверить кнопку чата → написать ИИ
2. Проверить источник трафика и платит ли за рекламу
3. Просканировать в ИИ Сканере
4. Сгенерировать демо-сайт
5. Задать контекстные вопросы → показать клиенту демо
6. Настроить виджет, проверить правильный email в профиле
7. И т.д. до передачи кода виджета клиенту

**Слабое место:** Только AI-консультант/виджет. Нет HR, финансов, маркетолога, юриста, отдела продаж. Платный вход для агентств ($500+/мес) — высокий порог.

## Ключевые UX-паттерны (взять в свой кабинет)

1. **Переключение контекста** (intly): одна учётная запись — два режима (агент / клиент своего бизнеса)
2. **Канбан лидов** (intly): встроен прямо в кабинет, не отдельная CRM
3. **Каталог интеграций** (intly): плитки по категориям, статус "подключено / не подключено"
4. **Финансовый блок партнёра** (оба): доход по дням, hold, реквизиты, выплаты — всё в одном месте
5. **ИИ Сканер** (b24u): отдельный раздел, история сканов, экспорт
6. **Сравнение тарифов** (b24u): большая таблица с ✓/✗ по каждой функции — конвертирует
7. **Пороги эскалации с визуализацией** (intly): 0–50% / 50–70% / 70–100% — наглядно
8. **Маркетплейс AI-инструментов** (suvvy): ready-to-use боты из каталога
